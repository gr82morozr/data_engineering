PUT my-index-000001
{
  "mappings": {
    "properties": {
      "status_code": { 
        "type":  "long"
      },
      "session_id": { 
        "type":  "long",
        "index": false
      }
    }
  }
}

[status_code] is regular field,which is indexed.
[session_id] is NOT indexed, it "doc-value-ONLY" 

it would be usefully for data aggregation, sorting.
